# Code of Conduct

## Our Pledge

We as contributors, maintainers, and community members pledge to make participation in the ATLAS project a welcoming, respectful, and collaborative experience for everyone.

We are committed to fostering an environment where people can share ideas, discuss architecture, report issues, review code, and contribute improvements without fear of harassment or discrimination.

Our shared goal is to build a high-quality framework through constructive collaboration and mutual respect.

---

# Our Standards

Examples of behavior that contribute to a positive environment include:

* Being respectful and considerate.
* Welcoming different perspectives and ideas.
* Giving and accepting constructive feedback.
* Focusing discussions on technical merit.
* Helping others learn and grow.
* Assuming good intentions.
* Writing clear and professional communication.
* Respecting the time and effort of other contributors.
* Supporting documentation as an essential part of the project.

Examples of unacceptable behavior include:

* Personal attacks.
* Harassment or intimidation.
* Discriminatory language or behavior.
* Trolling or deliberately disruptive conduct.
* Publishing private information without permission.
* Hostile or unconstructive communication.
* Repeatedly ignoring established project processes.

---

# Technical Collaboration

ATLAS values thoughtful engineering discussions.

Disagreement is a normal part of software engineering.

Contributors are encouraged to challenge ideas, architectures, and implementations—but never the people behind them.

Technical discussions should always remain respectful, objective, and evidence-based.

The best idea should win, regardless of who proposes it.

---

# Architectural Discussions

Architectural decisions are expected to be discussed openly.

Contributors should:

* explain their reasoning,
* consider alternatives,
* listen to feedback,
* remain open to changing their opinion.

Architecture reviews are intended to improve the project—not to criticize individuals.

---

# Code Reviews

Code reviews are collaborative.

Review comments should:

* focus on the code,
* explain concerns clearly,
* suggest improvements where appropriate,
* encourage learning.

Receiving review feedback should be viewed as an opportunity to improve both the implementation and the project.

---

# Project Principles

The ATLAS community values:

* Quality over speed.
* Maintainability over shortcuts.
* Simplicity over unnecessary complexity.
* Documentation alongside implementation.
* Long-term thinking over quick fixes.

These principles guide both technical decisions and community interactions.

---

# Enforcement Responsibilities

Project maintainers are responsible for promoting a healthy and respectful community.

They may:

* clarify project expectations,
* moderate discussions,
* request changes to contributions,
* reject contributions that do not meet project standards,
* take appropriate action when this Code of Conduct is violated.

All enforcement decisions should be fair, consistent, and transparent whenever possible.

---

# Reporting Issues

If you experience or witness behavior that violates this Code of Conduct, please contact the project maintainers.

Reports should include as much relevant information as possible.

All reports will be handled respectfully and, where appropriate, confidentially.

---

# Scope

This Code of Conduct applies to:

* the source code repository,
* issue discussions,
* pull requests,
* documentation,
* project communication channels,
* community spaces related to ATLAS.

---

# Attribution

This Code of Conduct is inspired by the Contributor Covenant (version 2.1) and has been adapted to reflect the engineering principles and collaborative values of the ATLAS Framework.

---

# Final Statement

ATLAS is built by people who care about software quality, architecture, and collaboration.

A respectful community is essential for creating a framework that remains maintainable, reliable, and enjoyable to work on for many years.

Thank you for helping make ATLAS a welcoming place for everyone.
