# Releases

This directory contains the official release documentation for ATLAS snapshots.

Each release consists of:

- Release Notes
- Snapshot Manifest

Every release is generated from a completed sprint and represents a stable milestone.

## Release Criteria

A release requires:

- ✅ Successful Type Check
- ✅ Successful Build
- ✅ Successful Tests
- ✅ Completed Documentation
- ✅ Architecture Decision Records updated
- ✅ Sprint Documentation completed
